How to install the project
1. get the code
```
git clone https://github.com/zizoujab/polyfill-demo.git
```
2. install dependencies
```
cd polyfill-demo
composer install
```
3. start docker services 
```
$docker-compose up -d 
$docker-compose ps 
```
